# darkhours

TopDown Prototyp Shooter
(with endless rooms)

"It's unfortunately only a prototype. I will add after my vacations enemys and more gameplay elements."

Controls
[W A S D] - movement
[Left Mouse Button] - shoot
[Right Mouse Button] - reload
[Mouse] - looking around
